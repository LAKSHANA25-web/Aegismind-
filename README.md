# AegisMind

AegisMind is an AI-based text classification web application.

## Features
- Input text prediction
- Machine learning model integration
- FastAPI backend
- React frontend

## Technologies Used
- Python (FastAPI)
- React.js
- Machine Learning (Scikit-learn)
- Git & GitHub

## How to Run

### Backend
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
npm start
